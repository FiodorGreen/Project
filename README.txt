Данный репозиторий содержит алгоритмы для решения задачи о нахождении числа многоугольников периметра 4n с равными высотой и шириной,
лежащими на целочисленной решетке. Для многоугольников периметра 2n без условия равенства ширины и высоты последовательность расчитана 
вплоть до n = 25 (OEIS: A002931). Репозиторий содержит 3 файла:
main.py - визуализация конфигураций многоугольников для данного 2n на языке Python с помощью модуля PyGame
help_func.py - функция, вычисляющая данные многоугольники и запоминающая
